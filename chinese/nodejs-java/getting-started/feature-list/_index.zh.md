##  **Aspose.3D 功能**
###  **一般特性**
- 完全用JavaScript编写，可在所有支持JavaScript的环境中工作
- 支持桌面GUI应用程序、网络应用以及控制台应用程序。
- JavaDoc格式的 API 引用
- Microsoft Windows 或基于Linux的操作系统。
