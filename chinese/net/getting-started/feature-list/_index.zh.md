Aspose.3D for .NET 功能-3D 文件操作和转换 API 内置于 C#。

##  **一般特性**
- 完全写在 C# 中，与 .NET 框架一起工作。
- 需要 .NET 环境。
- 支持 Windows 表单和ASP .NET 应用程序。
- API 以 HTML 和 Microsoft 帮助格式引用。
- 支持的 .NET 框架 (2.0、3.5、4.0、ClientProfile、4.5.0、4.5.1、4.6.0、4.6.2、4.7、4.7.2)。
- 32位操作系统支持。
- 64位操作系统支持。
