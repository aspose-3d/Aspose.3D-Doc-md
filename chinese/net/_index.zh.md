**欢迎使用 Aspose。3D for .NET API**

Aspose.3D for .NET 是一个类库项目，它使 .NET 应用程序能够自动连接 3D 格式，而无需在服务器上安装 3D 建模和呈现软件。Aspose.3D API 允许工程师读取、转换、构建、更改和控制 3D 文档格式的内容。

API 是用户友好的，并且比从头开始创建类似的解决方案节省时间和金钱。
