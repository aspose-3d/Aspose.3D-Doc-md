##  **Aspose.3D 功能**
###  **一般特性**
- 完全在 Java 中编写，因此适用于所有 Java 支持的环境
- 支持桌面GUI应用程序、网络应用以及控制台应用程序。
- JavaDoc格式的 API 引用
- 实时渲染和离屏渲染<sup>*</sup>.
- 支持JDK 1.8 - JDK 20



注意：渲染使用org.lwjgl.vulkan 3.2.1执行，需要JDK 9或更高版本（类版本为53），所有与渲染相关的API在JDK8版本中已被移除。
