Aspose.3D özellikleri


##  **Orted upported Platforms**

Platformlar Aspose.3D for Python via .NET Windows x64 veya x86 ve Python 3.5 veya daha sonra yüklü olan çok çeşitli linux dağıtımlarında kullanılabilir. Hedef linux platformuna ek gereksinimler var:
- GCC-6 çalışma zamanı kütüphaneleri (veya üstü)
- Dependencies of .NET Core Runtime. Installing .NET Core Runtime itself is NOT required
- For Python 3.5-3.7: The pymalloc build of Python is needed. The --with-pymalloc Python build option is enabled by default. Typically, the pymalloc build of Python is marked with m suffix in the filename.
- Libpython Python kütüphanesini paylaştı. -- Shared-paylaşılan Python yapı seçeneği varsayılan olarak devre dışı bırakılır, bazı Python dağıtımları libpython paylaşılan kütüphaneyi içermez. Bazı linux platformları için, libpython paylaşımlı kütüphane, paket yöneticisi kullanılarak kurulabilir, örneğin: sudo apt-get libpython3.7. Ortak sorun, libpython kitaplığının paylaşılan kütüphaneler için standart sistem konumundan farklı bir yere kurulmasıdır. Sorun, Python tutarındaki alternatif kütüphane yollarını ayarlamak veya paylaşılan kütüphaneler için sistem standart konumundaki libpython kütüphane dosyasına sembolik bir bağlantı oluşturarak sabitlemek için Python yapı seçeneklerini kullanarak düzeltilebilir. Tipik olarak, libpython paylaşılan kütüphane dosya adı libpythonx. ym. so.1.0 Python 3.5-3.7 veya libpythonx. y. so.1.0 Python 3.8 veya üstü için (örneğin: libpython3.7m.so.1.0, libpython3.9.so.1.0).
- Libgdiplus 6.0.1 veya üstü
