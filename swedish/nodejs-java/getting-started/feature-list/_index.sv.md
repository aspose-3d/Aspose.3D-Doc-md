##  **Aspose.3D Funktioner**
###  **Allmänna egenskaper**
- Skrivet helt i JavaScript och fungerar i alla JavaScript-stödda miljöer
- Stöder skrivbordsgränssnittsprogram, webbapplikationer och konsolprogram.
- API Referens i JavaDoc-format
- Microsoft Windows eller Linuxbaserat operativsystem.
