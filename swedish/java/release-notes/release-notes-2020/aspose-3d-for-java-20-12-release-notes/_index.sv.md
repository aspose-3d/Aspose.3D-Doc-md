## API ändringar ##

Denna version fixade en massa buggar, och minskade en hel del minnesfotavtryck samtidigt öka upp TriMesh prestanda.

Det finns vissa ändringar av gränssnittet, men dessa ändringar orsakar inte kodbrytning.
