## **Allmänna egenskaper**
- Skrivet helt i C#, fungerar med .NET Framework.
- .NET miljö behövs.
- Stöder Windows Formulär och ASP.NET ansökningar.
- API hänvisning i HTML och Microsoft Hjälpformat.
